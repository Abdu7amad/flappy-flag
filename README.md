# 🐦 Flappy Flag

Классическая игра **Flappy Bird**, но с флагами стран мира!  
Выбери флаг своей страны перед стартом и лети как можно дальше.

## 🚀 Как играть

1. Открой [Flappy Flag](https://Abdu7amad.github.io/flappy-flag) в браузере
2. Выбери свой флаг (включая 🇹🇯 Таджикистан)
3. Кликай / тапай / жми **Пробел**, чтобы взмахнуть
4. Пролетай между трубами и набирай очки!

## 📱 Установить как приложение на Android (PWA)

### Самый простой способ (рекомендуется):
1. Открой игру в **Chrome** на телефоне
2. Нажми на **⋮** (три точки) → **«Добавить на главный экран»**
3. Подтверди — иконка игры появится на рабочем столе как обычное приложение!

### Получить готовый .apk файл (для установки без браузера):

1. Перейди на сайт **[PWABuilder.com](https://www.pwabuilder.com/)**
2. Вставь ссылку на игру: `https://Abdu7amad.github.io/flappy-flag`
3. Нажми **Start** → **Generate**
4. Выбери **Android** → скачай готовый **.apk** файл
5. Установи на телефон (разреши установку из неизвестных источников)

Готово! Теперь у тебя настоящее приложение Flappy Flag.

## 🛠️ Для разработчиков

Репозиторий содержит:
- `index.html` — полностью автономная игра
- `manifest.json` — PWA манифест
- `sw.js` — Service Worker для работы оффлайн

Игра работает полностью оффлайн после первой загрузки.
